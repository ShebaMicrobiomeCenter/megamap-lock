# 🔐 MegaMap Lock - Encrypted Data Viewer

ממשק יפה להצפנה וצפייה בנתוני CSV מוצפנים באמצעות סיסמה.

## 📋 תיאור

זוהי אפליקציית ווב מאובטחת המאפשרת:
- ✅ הצפנה של קבצי CSV באמצעות **AES-256-GCM**
- ✅ צפייה בנתונים המוצפנים אחרי הזנת סיסמה
- ✅ שמירת סיסמה בטוח ב-localStorage (ניתן לניקוי)
- ✅ חיפוש וסינון בנתונים
- ✅ הורדה של נתונים כקובץ CSV
- ✅ ממשק responsive וידידותי למשתמש

## 🔒 אבטחה

- **אלגוריתם הצפנה:** AES-256-GCM
- **גזירת מפתח:** PBKDF2-SHA256 (200,000 iterations)
- **אורך Salt:** 16 בתים
- **אורך IV:** 12 בתים
- **Auth Tag:** 16 בתים
- **הפעלה:** Browser-side בלבד (כל הצפנה/פענוח קורה במכשיר)

## 🚀 התחלה

### 1️⃣ הצפנת קובץ

```bash
# התקנת תלויות (Node.js)
node encrypt.js latest-light.csv latest-light.csv.enc "your-secure-password"
```

הפקודה תיצור קובץ `.enc` המוצפן.

### 2️⃣ פתיחת הממשק

פשוט פתח את `index.html` בדפדפן:
```bash
# אפשרות 1: פתח ישירות
open index.html

# אפשרות 2: השתמש בשרת מקומי
python3 -m http.server 8000
# ואז גש ל: http://localhost:8000
```

### 3️⃣ צפייה בנתונים

1. בחר את הקובץ המוצפן (`.enc`)
2. הכנס את הסיסמה
3. לחץ על "פענוח וצפייה"
4. חפש, סנן, והורד נתונים לפי הצורך

## 📁 פורמט הקובץ המוצפן

הקובץ המוצפן מכיל:

```
[MEGAMAP1 header: 8 bytes]
[Salt: 16 bytes]
[IV: 12 bytes]
[Ciphertext + Auth Tag]
```

## 🌐 GitHub Pages + GitHub Action

### הגדרה מהירה

1. **פרסם ב-GitHub Pages:**
   - דחוף את הקובץ לריפו
   - הפעל GitHub Pages מהגדרות הריפו
   - הכתובת שלך תהיה: `https://[USERNAME].github.io/megamap-lock/`

2. **סנכרון אוטומטי מ-MEGAMAP:**
   - בחר ב-Settings > Variables
   - הוסף משתנים:
     - `MEGAMAP_OWNER` (בעל הריפו)
     - `MEGAMAP_REPO` (שם הריפו)
     - `MEGAMAP_FILE_PATH` (נתיב הקובץ)
   - GitHub Action ירוץ כל יום בחצות

📖 **קרא את [SETUP_GITHUB_ACTION.md](./SETUP_GITHUB_ACTION.md) להוראות מפורטות**

## 🛡️ הערות אבטחה

- **אין שמירה בשרת:** כל הצפנה/פענוח קורה **בלבד** בדפדפן המשתמש
- **סיסמה בטוחה:** לא נשלחת לשרת, רק משמשת לגזירת מפתח
- **localStorage:** ניתן להסיר ידנית בכל עת דרך כפתור "ניקוי סיסמה"
- **פתיחה מהקובץ:** אם אתה פוצח `index.html` ישירות (לא דרך שרת), הוא עובד בלי בעיות
